Classes - toda a estrutura de codigo na linguagem java é distribuido em arquivos como extensão .java denominados de classe, que será composto identificador , caracteristicas e comportamentos.

PRIMEIRO CRIA UM MODE , DPS CRIA O OBJETO 


classe de modelo - classes que representem estrutura de domininio de aplicação 

classe de serviço - classes que contem regras de negocio e validação de nosso sistema 

classe de repositorio - classes que contem uma integração com o banco de dados 

classe de controle (controller) - classes que possuem a finalidade de disponibilizar alguma comunicação externa a nossa aplicação 

classe utiitaria - classe que contem recursos comuns á toda nossa aplicação

Deixar os arquivos e pastas bem organizadas e fundamental , deixando bem explicita pelo nome daquela classe/pasta, pra facilitar futuramente , além de deixar bem organizado (ngm gosta de uma área de trabalho cheia de arquivos sem organização).
